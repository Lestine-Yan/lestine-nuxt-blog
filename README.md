这是一个基于 [Nuxt: The Progressive Web Framework](https://nuxt.com/) (v4.3.1) 开发的个人blog项目，项目开始之初，我还是一个在校的大一新生，我希望这个项目可以作为我前端的起点，在blog的搭建中学习和成长。  

2026.4.14

网站如今已经正式上线部署在云服务器上，开始的时候是想设计一个很多动效很炫酷的那种blog，但是由于我的美术实在是不行，技术力也很薄弱，所以网站如今是现在的这个样子。

用了比较取巧的方式，直接拿了张我比较喜欢的一个图当背景，并且以这个背景的颜色作为主题色，算是填补了一下技术力和美术设计的缺陷。  

但是，显然这是远远不够的，可以预见的一天，我应该会把这个blog网站再重构一遍，重新设计一次ui布局，更有科技感的美术等等.....
